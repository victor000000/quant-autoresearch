What this is

An autonomous research loop that invents and back-tests single-ticker ETF trading strategies on real market data (QuantConnect), and keeps only the ones that beat buy-and-hold and survive strict out-of-sample + multiple-testing checks. The AI proposes and runs the experiments; a human steers.

Bottom line today — durable single-ticker alpha is scarce: only GLD (gold trend-following, Calmar ~2.1631) and UUP (US-dollar regime, ~1.0831) beat buy-and-hold in a way that survives the deflation (multiple-testing) gate — every other ETF is best held passively. Each round the loop attacks the weakest-Calmar ticker with two competing hypotheses and keeps a winner only if it clears every honest gate. The backtest is verified leak-free and fully online, using only models trained in the cloud (see deploy).

How to read the numbers
  • Calmar = annual return ÷ worst drawdown (higher is better; above 3 is strong).
  • MaxDD (MDD) = the deepest peak-to-trough loss along the way.
  • CAGR = compounded annual return. Sharpe = return per unit of volatility.
  • Edge = how much a strategy beats simply buying and holding the same ETF.
  • “Survives deflation” = the result still looks real after correcting for how many strategies we tried (a guard against luck/overfitting). A weak edge that was lucky over a short window is rejected.
  • Recipe / cell = the exact pipeline that produced a result: asset · bar-clock · labeling-method · sizing · threshold.

The research arc

How the project got here, in four acts — from a failed start to the rule that now governs the whole board.

I · The null

Single-asset long-only ML couldn't beat buy-and-hold across 6 rounds → reframed to a v2 per-ETF tournament.

6-round NULL → tournament
II · The unlock

Long-only can't beat a declining asset; shorting + a directional label + ③ mean-reversion features unlocked TLT.

TLT 0.31 → 1.52
III · The correction

A bar-threshold look-ahead (full-series stats incl. OOS) was inflating results; the leak fix re-validated the whole board.

XLE 2.26 → 0.64
IV · Convergence

One rule governs the board — time when the hold is weak, hold when the trend is strong (f_timing_when).

7/7 at leak-free ceilings

Top insights

The most important things learned, each with the evidence that backs it.

  1. 01Meta-labeling crosses Calmar>3
  2. 02Causal mean-reversion features unlock mean-reverting ETFsTLT ③
    Adding causal RSI + Bollinger-%b oscillators (③) took TLT from 0.31 → 1.52 Calmar. Control (same axis/labeler, no features) stays at −0.14, so the lift is the features, not leakage.
  3. 03Ensembling two comparable labelers lifts; one dominant signal dilutesIWM R59
    Averaging calibrated triple_barrier + BGM predictions lifted IWM 0.93 → 1.14, but dilutes TLT (one signal already dominates). Ensemble only when two legs are individually strong.
  4. 04No-lookahead invariant: never filter trading bars by a future labelG3 audits
    Two enforced rules: (1) test/val predictions cover EVERY causal bar — a future label may filter only the FIT set, never which OOS bars trade; (2) bar thresholds fit on TRAIN minutes, not the full series (a magnitude-axis full-series threshold leaked OOS into bar placement — found + fixed).
  5. 05Selectivity beats exposure when directional accuracy is thinIWM R59
    On IWM's ensemble a tighter entry threshold (0.45) beat a looser one (0.40, no vol-throttle): +1.14 vs +0.94. Trade less, trade better.
