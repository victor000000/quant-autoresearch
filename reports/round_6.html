1. Hypothesis

Round 5 showed the label-independent inverse-vol overlay washes out the labeler. Turn it off (VOL_FLOOR=1.0 ⇒ $g\equiv1$) so sizing is purely label-driven, $w=\texttt{cdf\_bet}(p,\tau)$, and the position goes flat when $p\le\tau$. Prediction: always_long reverts to ~buy-hold while a real labeler (kmeans2stage) that sometimes predicts down diverges from it.

2. Method

Dollar axis (held constant vs R5 to isolate the overlay on→off change) × {kmeans2stage, carry, always_long} × 7 ETFs, 2-node parallel. $w=\texttt{cdf\_bet}(p,\tau)=\big[2\Phi(z)-1\big]_0^1,\ z=(p-\tau)/\sqrt{p(1-p)}$, gate $w=0$ if $p\le\tau$. Identical in trainer (VAL) and infer (OOS). Splits standard.

4. Verdict & the chapter-closing finding

The hypothesis is false: with the overlay off the position does not go flat, because on these up-trending ETFs the calibrated $p$ is almost always $>\tau$ — the model predicts "up" nearly everywhere — so $w>0$ for the whole OOS window → one entry, held → buy-hold, identical for every labeler.

Overlay ON (R5): active but the overlay drives trades → label washed out. Overlay OFF (R6): the model is always-long → buy-hold → label washed out. Either way ② labeling cannot express itself on a single trending long-only asset.

Six rounds converge: single-asset, long-only ML on trending ETFs ≈ buy-hold; axis/labeler/sizing are all washed out because the model predicts "up" almost always. This experimentally confirms the archive's lesson rather than assuming it: best single ≈ 1.6, and the only path past it (best pair 2.91; G1>3.0) is cross-asset structure. TLT −0.06 (negative control, as expected).
